# 🏠 House Price Prediction using Multiple Linear Regression

## 📌 Project Overview

This project focuses on building a **Machine Learning model** to predict house prices based on various features such as area, year built, and neighborhood.

The model uses **Multiple Linear Regression** to analyze historical housing data and generate accurate price predictions.

---

## 🎯 Objectives

* Predict house prices using structured data
* Perform data preprocessing and feature engineering
* Visualize relationships between variables
* Evaluate model performance using statistical metrics

---

## 🛠️ Tech Stack

* **Python**
* **Pandas** – Data manipulation
* **Seaborn** – Data visualization
* **Scikit-learn** – Machine learning model

---

## 📂 Dataset

The dataset contains information such as:

* Lot Area
* Year Built
* Sale Price
* Neighborhood

> Note: Dataset should be placed in the project directory or update the file path accordingly.

---

## ⚙️ Project Workflow

### 1. Data Collection

* Import dataset using Pandas

### 2. Data Preprocessing

* Handle missing values
* Convert categorical variables into numerical format using one-hot encoding

### 3. Data Visualization

* Scatter plots to understand relationships between variables

### 4. Model Building

* Split dataset into training and testing sets
* Train a **Linear Regression** model

### 5. Prediction

* Predict house prices using test data

### 6. Evaluation

* R² Score
* Adjusted R² Score

---

## 📊 Model Evaluation Metrics

* **R-squared (R²):** Measures how well the model explains variability
* **Adjusted R²:** Adjusts R² based on number of features

---

## ▶️ How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/your-username/house-price-prediction-ml-regression.git
```

2. Navigate to the project folder:

```bash
cd house-price-prediction-ml-regression
```

3. Install dependencies:

```bash
pip install pandas seaborn scikit-learn
```

4. Run the script:

```bash
python House_Price_Prediction.py
```
